## 各仮説の妥当性評価

### 仮説①：6ヶ月以上同値 → 調査を受け入れていない

**→ 妥当性：高い。ただし「受け入れていない」の解釈は細分化が必要。**

**統計的根拠：**

V/D/A がそれぞれ 0〜18 の範囲を取るとして、3次元が6ヶ月連続で完全同値になる確率は：

- 一様分布を仮定した場合：**約 6.6 × 10⁻²⁰**（事実上ゼロ）
- 最頻値（9）が回答確率 20% という偏分布を仮定しても：**約 3.3 × 10⁻¹¹**

これは統計的に「偶然の一致」ではありえず、**意図的または習慣化した応答行動**である。

**さらに重要な事実：** 12名中 **12名全員（100%）が V=D=A**（三次元が同値）。UWES-9 は活力・熱意・没頭を**独立した構成概念**として設計されており、本来は三次元がほぼ同値になることは稀である。これは三次元を区別して内省せず、一律の値を入力していることを強く示唆する。

また全データ（1,406行）中 V=D=A の割合は **28.3%** と、そもそも高い。長期固定に至る前の段階から、V=D=A で回答する傾向を持つメンバーが多い。

**「受け入れていない」の解釈について：**

「調査を拒否している」という積極的な意思だけでなく、心理測定学で確立されている **satisficing（満足化）** や **careless responding（不注意な回答）**（Krosnick, 1991）として解釈する方が精確なケースを含む。すなわち「拒否の意思」と「無関心・省力化」を区別して評価する必要がある。これは後述の Level 別解釈に直結する。

### 仮説②：原因は「内面を知られたくない」「調査に意味がない」等の拒否姿勢

**→ 妥当性：可能性としては有力だが、他の解釈も排除できない。**

データで確認できる点と限界：

- 固定化への転換タイミングが特定できるメンバー（小川、鈴木、吉田など）では、固定化**前**にも変動があった。変動できる状態から意図的に固定した、という読み方は可能。
- 一方、**橘 直宏（19ヶ月、初回から固定）** のように、最初から一貫して 9/9/9 というケースは、回答負荷を最小化する **省力的応答スタイル**（=調査への積極的拒絶というより無関心）の可能性が高い。
- **根津 蓮太郎** は固定前 6ヶ月では V≠D≠A で応答しており（V=D=A率 0%）、その後 6/6/6 に固定した。これは状況の変化への反応として特定値に固着した可能性がある。

**排除できない代替解釈：**

- 入力インタフェースの問題（前回値が表示されていてそのまま送信している）
- 調査疲れ（fatigue）による省力化
- 真剣に考えた結果として毎月同じ値になっている（ただし統計的には極めて低確率）

### 仮説③：期間が長いほど拒否意思が強い

**→ 妥当性：概ね妥当。ただし「意思の強さ」より「行動の固着度」として定義する方が適切。**

行動科学的には、同じ行動が繰り返されるほど習慣化・自動化し、変容コストが高まる（Lally et al., 2010, *European Journal of Social Psychology*）。これは「意思が強い」というより「変えにくい状態になっている」という解釈の方が介入戦略の設計に有効。

**介入優先度への組み込み論拠：** 期間が長いほど、通常の engagement 改善策では効果が出にくく、**まず調査への姿勢そのものに働きかける必要がある**という点で、個別介入の優先度は高まる。

### TYPE_B_HIGH が別扱いである根拠

**鈴木 尚宜：Thriving(54)からの意図的格下げ**

**吉田 雄一：真正回答から平均値への収束固定**

## intervention_priority への組み込み方針の修正提案

前回提案した単純な `streak_score × level_risk` に対して、以下の修正列を追加することを推奨する：

**`resistance_subtype`（文字列ラベル）**

| 値                | 対象                                 | 介入方針                             |
| ----------------- | ------------------------------------ | ------------------------------------ |
| `HIGH_AVOIDANCE`  | 鈴木, 吉田                           | 調査目的の透明性、管理への不信解消   |
| `MID_EVASION`     | 立田, 小川, 中津川, 三倉, 藤原, 根津 | 調査参加意義の再確認、心理的安全性   |
| `ZERO_FIXED`      | 武藤                                 | 在籍確認優先、exit interview 相当    |
| `PHASE_SHIFTED`   | 三塩                                 | 転換トリガーの把握、ゼロ期の不満確認 |
| `FROM_FIRST_WAVE` | 橘                                   | 最初から省力化スタイルの可能性       |

**`prior_zero_flag`（boolean）**

ゼロ固定期間が過去に存在したかどうかを示すフラグ。三塩のような TYPE_C を TYPE_B_MID と混同しないための識別子。`prior_zero_months`（ゼロ固定だった月数）をセットで持たせると、score 補正に使える。

**score 補正ルール（追加）：**

- `HIGH_AVOIDANCE` かつ 現在 High 以上 → `level_risk` を 2 に上方修正（調査不信リスクを加味）
- `PHASE_SHIFTED`（TYPE_C）→ `survey_resistance_score` に `prior_zero_months / 3` を加算（過去の累積抵抗を引き継ぐ）

これにより、三塩の実態スコアは 2（現行）→ **2 + 12/3 = 6** となり、Moderate 中程度の固定者と同等以上のリスクとして扱われることになる。

**3名のそれぞれが本質的に異なる種類の問題**を持っており、一括して「3ヶ月固定」として扱えない点が重要。

**加部 正悟（全ゼロへの収束）**

過去19ヶ月は V≠D≠A で真正に変動していた回答者。直近3ヶ月で 0/0/0 に固定した。固定直前も Critical レベルへの一方的な低下が続いており、「調査拒否」よりも**エンゲージメントが本当に崩壊している**可能性が同時に存在する。6ヶ月閾値を待たずに早期検出すべき対象である点は明確だが、解釈は全ゼロ固定（武藤相当）として扱い、「真の低下 vs. 拒否」の判別を面談で確認する必要がある。

**天辰 貴志（間欠的固定 / リセット問題）**

これが 3ヶ月閾値を議論する上で最も本質的な問題を示す。履歴は：

- 9/9/9 を**8ヶ月固定**（2025-03〜10）→ flag_constant_6m=TRUE として当時検出済み
- 2025-11 に一度だけ 28/10/9/9 に逸脱
- その後また 9/9/9 に戻り現在3ヶ月固定中

「1ヶ月の逸脱でカウントが6からゼロにリセットされる」という現行ロジックの構造的な問題がここに露呈している。3ヶ月閾値に変更しても、この問題は 6→3 に縮まるだけで根本的な解決にはならない。

**内田 賢（ノイズ事例）**

データに同一 wave の重複エントリが存在し、現在の固定値は V=9, D=8, A=8 で V≠D≠A。9/9/9 ですらない。過去にも 3ヶ月ストリークが2回あり両方とも自然終了。3ヶ月閾値では誤検出になる可能性が高い。

------

## 結論：閾値の数値変更よりも構造的な修正が有効

3ヶ月に変更しても、**評価方法の精度向上にはならず、むしろ誤検出（内田）を増やすリスクの方が大きい**。本質的な問題は以下の2点であり、どちらも閾値変更では解決しない：

**問題①：カウントリセット問題（天辰 貴志型）**

「1ヶ月の逸脱でストリークカウントがゼロに戻る」仕様のため、短期の揺らぎで長期の固定パターンが見えなくなる。対処は閾値変更ではなく、**`max_streak_ever`（過去の最長固定期間）と `prev_constant_flag`（過去に flag_constant_6m が立ったことがあるか）を新列として保持**することで対応できる。

**問題②：全ゼロ収束の早期検出（加部 正悟型）**

Critical レベルへの真の低下と調査拒否を区別しつつ、早期に介入する必要がある。これは閾値の問題ではなく、**全ゼロ固定については閾値を3ヶ月に短縮**し、その他の値（9/9/9等）は6ヶ月のまま維持するという **Level 別の閾値差分設定**として扱うべきだ。

具体的には：

| 固定値の種類             | 推奨閾値  | 理由                                                       |
| ------------------------ | --------- | ---------------------------------------------------------- |
| 全ゼロ（0/0/0）          | **3ヶ月** | 加部・武藤型。放置するほど介入が遅れる                     |
| Critical レベル（E<3）   | **3ヶ月** | 同上                                                       |
| Low〜Moderate（9/9/9等） | **6ヶ月** | 3〜5ヶ月の自然終了率が54%あり、早期検出の意味が薄い        |
| High 以上（12/12/12等）  | **6ヶ月** | 鈴木・吉田型。短期では「高い値が続いている」に見えてしまう |

また、天辰型に対しては `max_streak_ever >= 6` または `prior_flag_constant == True` のフラグを持たせ、1ヶ月の逸脱後でも注意対象として維持することを推奨する。

## 分析結果：離脱組15名の共通特徴

### 最重要の発見：「共通の特徴はない」が正確な答え

離脱組15名を定量的に比較した結果、最終的な engagement レベルや直前トレンドは**継続組と統計的に区別できない**。

| 指標                      | 離脱組（15名） | 継続組（90名） | 差                       |
| ------------------------- | -------------- | -------------- | ------------------------ |
| 最終 engagement 平均      | 25.9           | 24.2           | +1.7（離脱組の方が高い） |
| 全期間 engagement 平均    | 24.6           | 24.1           | +0.5                     |
| 直前3ヶ月 trend：**上昇** | **8名（53%）** | —              | —                        |
| 直前3ヶ月 trend：横ばい   | 5名（33%）     | —              | —                        |
| 直前3ヶ月 trend：低下     | 2名（13%）     | —              | —                        |

離脱者の53%が**最終3ヶ月は上昇傾向**であり、「エンゲージメントが低い・低下しているから離脱した」という単純な解釈は成立しない。

------

### 共通しているのは「変動性の高さ」

唯一、継続組と数値的な差が現れたのは**engagement の変動性（標準偏差）**だった。

|                     | 離脱組              | 継続組           |
| ------------------- | ------------------- | ---------------- |
| std 平均            | **5.14**            | **3.08**         |
| std 中央値          | 3.42                | 2.59             |
| std ≥ 5.0（高変動） | **4名/15名（27%）** | 10名/90名（11%） |
| std ≤ 1.0（低変動） | 3名/15名（20%）     | 12名/90名（13%） |

離脱組の高変動率（27%）は継続組（11%）の約2.5倍。ただし**n=15 という小標本のため統計的有意性の判断には慎重さが必要**であり、傾向として読む水準に留まる。

------

### 類型別の読み方

実態は5つの異なるパターンに分かれており、一括できない。

**① 観測期間不足（3名：森、加治、塚田）**

1〜3波しかデータがなく、パターンを読み取ること自体が不可能。これは「早期離脱」の問題であり、エンゲージメントの軌跡の問題ではない。

**② 高エンゲージメントで突然消滅（2名：金 志勇、金 璣夏）**

High〜Thriving レベルで安定または上昇しており、直前に何らかの問題シグナルは出ていない。エンゲージメント問題ではなく、**昇進・転職・組織都合の異動**といった外部要因が支配的と推測される。この2名は「介入の失敗」ではなく「サーベイの対象外となった」と読む方が自然。

**③ 低下傾向の末の離脱（3名：髙﨑、田沢、李 誠鎬）**

最もわかりやすい「典型パターン」。intervention_neg が高い時期があり、低下傾向が継続した末に離脱。ただし3名しかおらず、これが離脱組の代表的パターンとは言えない。

**④ 高変動・不安定で離脱（3名：又賀、永元、神井）**

このグループが最も示唆的。**直前に回復傾向を示しながら消滅**している点が重要で、一時的な数値回復が必ずしも安定を意味しないことを示す。

又賀 秀輝の履歴が典型例：E が 0→35→0→13→17 という激しい乱高下を繰り返し、最後は17（Moderate）という比較的良い値で消えた。この「数値が正常に見えるが実態は不安定」というパターンは、現行の `intervention_priority` では捉えにくい。変動性の高さ（std=9.6）が唯一のシグナルだった。

**⑤ 調査拒否固定（1名：武藤）**

全ゼロ18ヶ月固定という特殊パターン。

**⑥ 安定・横ばいで消滅（4名：大﨑、曽田、李 炅大、森）**

engagement の観点からは問題シグナルが出ていない。組織都合の異動や配置転換が主因と推測される。**この離脱はサーベイで予測・防止できる類のものではない。**

------

### intervention_priority への示唆

離脱の事前シグナルとして機能しうるのは、現行指標の中では**変動性（E_std）の高さだけ**であり、かつそれも十分ではない。

特に④の高変動・直前回復パターンは、現行の `trend_refined` や `intervention_priority_neg` では「上昇期待」「回復」と判定されてしまい、むしろ**安全サインとして誤って読まれる危険がある**。

追加できる指標として、`E_std_all`（全期間変動性）または直近6〜12ヶ月の変動レンジを `instability_flag` として持たせ、「現時点は回復方向だが変動性が慢性的に高い」状態を識別することが有効と考えられる。ただし、この指標単体では**退職と異動（組織都合）を区別することはできない**という限界は残る。
