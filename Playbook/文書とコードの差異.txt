docs と現行コード (we_analyzer.py) の差異

  1. 定数値の差異

  ┌─────────────────┬───────┬─────────────┬────────────────────────────────┐
  │                 │ docs  │ コードの実  │                                │
  │      定数       │ の記  │   際の値    │              備考              │
  │                 │ 載値  │             │                                │
  ├─────────────────┼───────┼─────────────┼────────────────────────────────┤
  │ BIG_CHANGE_PERS │ 2.0   │ 2.4         │ docs は「2σ基準」、コードは    │
  │ ONAL_Z          │       │             │ 2.4σ                           │
  ├─────────────────┼───────┼─────────────┼────────────────────────────────┤
  │ TREND_SLOPE_STD │ 0.45  │ 0.55        │ コードでは15%タイルに対応する  │
  │                 │       │             │ 値に変更済み                   │
  └─────────────────┴───────┴─────────────┴────────────────────────────────┘

  2. trend_base が参照する標準化傾き

  ┌────────┬───────────────────────┬────────────────────────┐
  │        │ docs (raw_indicators) │      コードの実装      │
  ├────────┼───────────────────────┼────────────────────────┤
  │ 使用列 │ E_slope_6_std_12      │ E_slope_6_std_6 (L950) │
  └────────┴───────────────────────┴────────────────────────┘

  docs では E_slope_6_std_12 で判定すると記載されていますが、コードでは
  E_slope_6_std_6 を使用しています。

  3. trend_refined の条件差異

  ┌───────────┬─────────────────────┬──────────────────────────────────────┐
  │ パターン  │    docs (最終版)    │              現行コード              │
  ├───────────┼─────────────────────┼──────────────────────────────────────┤
  │ 上昇継続  │ trend_recent in     │ trend_recent in ["上昇", "急上昇",   │
  │ (P2)      │ ["上昇", "横ばい"]  │ "連続上昇", "横ばい"] + E_delta_1 >= │
  │           │                     │  0                                   │
  ├───────────┼─────────────────────┼──────────────────────────────────────┤
  │ 低下継続  │ trend_recent in     │ trend_recent in ["下降", "急落",     │
  │ (P2)      │ ["下降", "横ばい"]  │ "連続下降", "横ばい"] + E_delta_1 <= │
  │           │                     │  0                                   │
  ├───────────┼─────────────────────┼──────────────────────────────────────┤
  │ 回復 (P4) │ abs(E_slope_6) >    │ E_slope_6 チェックなし               │
  │           │ TREND_SLOPE 必要    │                                      │
  ├───────────┼─────────────────────┼──────────────────────────────────────┤
  │ 低下危機  │ abs(E_slope_6) >    │ E_slope_6 チェックなし               │
  │ (P4)      │ TREND_SLOPE 必要    │                                      │
  ├───────────┼─────────────────────┼──────────────────────────────────────┤
  │ 上昇期待  │ abs(E_delta_1) >    │ delta チェックなし（trend_recent     │
  │ (P5)      │ TREND_DELTA (1.0)   │ のみ）                               │
  │           │ 必要                │                                      │
  ├───────────┼─────────────────────┼──────────────────────────────────────┤
  │ 低下警戒  │ abs(E_delta_1) >    │                                      │
  │ (P5)      │ TREND_DELTA (1.0)   │ delta チェックなし                   │
  │           │ 必要                │                                      │
  ├───────────┼─────────────────────┼──────────────────────────────────────┤
  │ 低下懸念  │ abs(E_delta_1) >    │                                      │
  │ (P6)      │ TREND_DELTA (1.0)   │ E_delta_1 < 0 のみ（閾値なし）       │
  │           │ 必要                │                                      │
  ├───────────┼─────────────────────┼──────────────────────────────────────┤
  │ 回復期待  │ abs(E_delta_1) >    │                                      │
  │ (P6)      │ TREND_DELTA (1.0)   │ E_delta_1 > 0 のみ（閾値なし）       │
  │           │ 必要                │                                      │
  └───────────┴─────────────────────┴──────────────────────────────────────┘

  4. change_tag の比較演算子

  ┌────────┬────────────────────────────────┬───────────────────────────────┐
  │        │              docs              │            コード             │
  ├────────┼────────────────────────────────┼───────────────────────────────┤
  │ 判定式 │ abs(E_delta_1) / E_std_12      │ abs(E_delta_1) / E_std_12     │
  │        │ **>=** BIG_CHANGE_PERSONAL_Z   │ **>** BIG_CHANGE_PERSONAL_Z   │
  └────────┴────────────────────────────────┴───────────────────────────────┘

  docs では >=（以上）ですが、コードでは >（より大きい）です。
