# 個人分析ロジック（最新版）

本ドキュメントは `we_analyzer.py` の実装と同期しています。  
計算対象は 1 名 × 1 調査月（= 1 行）です。

## 基本指標

- **E** … 当月の総エンゲージメント（V + D + A、最大 54）
- **E_delta_1** … 当月 − 前月
- **E_delta_1_prev** … 前月 − 前々月
- **E_momentum_3** … 直近 3 回平均 − その前 3 回平均
- **E_slope_6** … 直近 6 回の Theil–Sen 傾き（中央値勾配）
- **E_std_6** … 直近 6 回の標準偏差
- **E_min6_past / E_max6_past** … 当月を除く直近 6 回の最小 / 最大

`mid_window` は 6 を既定値とし、「直近 6 ヶ月」を示します。  
個人の回答件数が **MID_MIN_RECORDS = 3** 以下の場合は中期判定ロジック（傾き・安定性など）を行わず、関連指標はブランクになります。

## Trend_B_base（基礎トレンド）

| ラベル | 条件 |
| ------ | ---- |
| 未評価 | 回答履歴が MID_MIN_RECORDS 以下 |
| 上昇中 | E_slope_6 ≥ **+0.35** |
| 低下中 | E_slope_6 ≤ **−0.35** |
| 安定   | 上記以外 |

## Trend_B_recent（直近判定）

- **強い増減** を `E_delta_1 ≥ +5.0` / `E_delta_1 ≤ -5.0` と定義し、`E_delta_1_prev` も同じ閾値で評価する。
- **強いモメンタム** を `E_momentum_3 ≥ +1.5` / `E_momentum_3 ≤ -1.5` と定義する。

| ベース | 判定 | 条件 |
| ------ | ---- | ---- |
| 未評価 | 上昇 | 強い増加のみ |
| 未評価 | 下降 | 強い減少のみ |
| 安定   | 上昇 | 強い増加 かつ `E_delta_1_prev ≥ 0`（もしくは強いモメンタム） |
| 安定   | 下降 | 強い減少 かつ `E_delta_1_prev ≤ 0`（もしくは強いモメンタム） |
| 上昇中/低下中 | 上昇 | 強い増加 かつ (強いモメンタム または 連続で強い増加) |
| 上昇中/低下中 | 下降 | 強い減少 かつ (強いモメンタム または 連続で強い減少) |
| いずれも該当せず | 横ばい | 上記のどれにも該当しない |

## Trend_B_refined（優先判定）

下表を **優先度順** に評価し、最初に該当したラベルを採用します。  
表中に明記していない補足条件はすべて `Trend_B_base` / `Trend_B_recent` の値を指します。

| 優先 | ラベル   | 主な条件 |
| ---- | -------- | -------- |
| 1 | 上昇加速 | 上昇中 × 直近上昇、E_slope_6 ≥ **+0.35**、E_delta_1 ≥ **+5.0**、かつ (E_momentum_3 ≥ **+1.5** または 直前も強い増加) |
| 2 | 上昇継続 | 上昇中 × 直近横ばい、E_slope_6 ≥ **+0.35**、\|E_momentum_3\| < **1.5**、\|E_delta_1\| < **5.0**（上昇中の残りも最終的にここへ収束） |
| 3 | 低下懸念 | 上昇中 × 直近横ばい、E_delta_1 < 0 |
| 4 | 悪化 | 上昇中 × 直近下降、E_slope_6 ≥ **+0.35**、E_delta_1 ≤ **−5.0**、かつ (E_momentum_3 ≤ **−1.5** または 直前も強い減少)、E ≥ E_min6_past |
| 5 | 低下危機 | 上昇中 × 直近下降、上記悪化条件に加え **E < E_min6_past** |
| 6 | 低下加速 | 低下中 × 直近下降、E_slope_6 ≤ **−0.35**、E_delta_1 ≤ **−5.0**、かつ (E_momentum_3 ≤ **−1.5** または 直前も強い減少) |
| 7 | 回復期待 | 低下中 × 直近横ばい、E_delta_1 > 0 |
| 8 | 低下継続 | 低下中 × 直近横ばい、E_slope_6 ≤ **−0.35**、\|E_momentum_3\| < **1.5**、\|E_delta_1\| < **5.0**（低下中の残りも最終的にここへ収束） |
| 9 | 回復 | 低下中 または (安定かつ過去傾き ≤ **−0.35**) × 直近上昇、E_delta_1 ≥ **+5.0**、(E_momentum_3 ≥ **+1.5** または 直前も強い増加)、かつ E ≤ E_max6_past |
| 10 | 復活 | 上記回復条件に加えて **E > E_max6_past** |
| 11 | 上昇期待 | 安定 × 直近上昇、**−0.35 < E_slope_6 < +0.35**、E_delta_1 > 0、(E_momentum_3 ≥ **+1.5** または `E_delta_1_prev ≤ 2.0`) |
| 12 | 低下警戒 | 安定 × 直近下降、**−0.35 < E_slope_6 < +0.35**、E_delta_1 < 0、`E_delta_1_prev ≥ 0`、かつ (E_delta_1 ≤ **−5.0** または E_momentum_3 ≤ **−1.5**) |
| 13 | 安定維持 | 上記いずれにも該当しない安定ベースのケース（その他） |

> **補足**  
> 判定には `E_min6_past` / `E_max6_past` が必要です。これらは「当月を除く直近 6 回」の min / max を個人内で求めた値です。過去データが 6 件未満の場合は比較可能な値が得られるまで `NaN` となり、該当条件はスキップされます。

## C_stability（6 ヶ月の安定性）

| 優先 | ラベル   | 条件 |
| ---- | -------- | ---- |
| 1 | 不変     | 直近 `mid_window` (=6) ヶ月の **E, V, D, A がすべて同一値** |
| 2 | 安定     | E_std_6 ≤ **1.0** かつ \|E_momentum_3\| < **0.5** |
| 3 | 不安定   | E_std_6 ≥ **2.5** |
| 4 | やや安定 | 上記いずれにも該当しない場合 |

> **補足**  
> MID_MIN_RECORDS 以下の場合は `C_stability` を空文字にする（評価保留）。

安定性ラベルも優先順で評価します。

## Level_A（絶対水準）

E（0〜54）の絶対値に基づき、以下の順で分類します。

| ラベル      | 条件 |
| ----------- | ---- |
| Thriving    | E > **43** |
| Critical    | E < **3** |
| High        | E > **32** |
| Low         | E < **11** |
| Moderate    | 上記以外（11 以上 32 以下） |

## ChangeTag

- `変化大` … \|E_delta_1\| ≥ **6.0**
- 空文字 … 上記以外

## 参考

- 個人短期・中期の強み/弱みは、各人の伸長平均との差分位（expanding quantile）とロバスト Z 値を用いて判定します（`overwrite_short_mid_personal` 参照）。
- 部門集計（`DeptDashboard_MS`）は従来どおりセクション/グループ単位のパーセンタイルによるリスク判定を行っています。
